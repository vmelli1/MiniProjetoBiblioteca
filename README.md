# Mini Projeto Biblioteca 
Este ‚ um projeto em Java para cadastro de livros em uma biblioteca simples. 
 
## Funcionalidades 
- Adicionar livros com nome, ano, autor e categoria 
- Listar todos os livros cadastrados 
- Listar somente autores 
- Listar somente categorias 
- Sair do programa 
 
## Como Executar 
1. Abra o projeto em sua IDE Java favorita. 
2. Execute projetodebiblioteca.java. 
3. Utilize o menu para interagir com o programa.
